Privacy Policy — ConsoleMind - AI

Last updated: July 7, 2025

Overview

ConsoleMind - AI is a browser developer tool extension that captures console error messages and provides explanations and fix suggestions. We are committed to protecting your privacy.

Data Collection

We do not collect any personal data.

The extension captures only JavaScript console error messages and stack traces from web pages you visit. This data is:

  • Stored locally on your device using Chrome's storage API
  • Never transmitted to our servers
  • Never shared with third parties
  • Automatically cleared when you close the browser tab

Third-Party Services (Optional)

If you choose to enable AI-powered explanations, the extension may send error messages to:

  • StackOverflow API — to search for relevant answers (public API, no auth required)
  • GitHub API — to search for related issues (public API)
  • MDN Web Docs API — to find documentation (public API)
  • OpenAI API — only if you provide your own API key
  • Anthropic API — only if you provide your own API key

These services are only contacted when you explicitly interact with the AI Assistant feature. No data is sent automatically in the background.

API Keys

If you provide an OpenAI or Anthropic API key, it is stored locally in your browser's extension storage. It is never transmitted to any server other than the respective API provider (OpenAI or Anthropic) when you make a query.

Permissions

The extension requests the following permissions:

  • Host permissions (all URLs) — Required to inject error-capture scripts on any website you visit. Only console error messages are captured, not page content.
  • Storage — To store captured errors and your preferences locally.
  • Notifications — To alert you when error spikes are detected.
  • Tabs — To identify which tab errors come from.
  • Scripting — To inject the error capture scripts into web pages.

Data We Do NOT Collect

  • Personal information (name, email, address)
  • Browsing history or web activity
  • Page content (text, images, forms)
  • Cookies or authentication tokens
  • Location data
  • Financial information
  • Keystrokes or mouse movements

Data Retention

Captured error data is stored locally and is automatically removed when the browser tab is closed. You can manually clear all data at any time using the "Clear" button in the extension.

Children's Privacy

This extension is a developer tool and is not directed at children under 13. We do not knowingly collect data from children.

Changes to This Policy

We may update this privacy policy from time to time. Changes will be reflected in the "Last updated" date above.

Contact

If you have questions about this privacy policy, please create an issue on our GitHub repository or contact us through the Chrome Web Store listing.
